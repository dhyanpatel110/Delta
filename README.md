## Project setup

### Clone this repo

```
git clone https://github.com/dhyanpatel110/Delta.git
```

### Install dependencies

```
npm install
```

### Compiles and hot-reloads for development

```
npm run start
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
